# Installation

Clone Repository

```bash
git clone https://github.com/yourusername/romi-traders.git
```

Open Project

```bash
cd romi-traders
```

Install Packages

```bash
npm install
```

---

# Environment Variables

Copy

```bash
.env.example
```

to

```bash
.env
```

Then update all values.

---

# Database

Generate Prisma Client

```bash
npx prisma generate
```

Create Migration

```bash
npx prisma migrate dev
```

Push Database

```bash
npx prisma db push
```

Open Prisma Studio

```bash
npx prisma studio
```

---
